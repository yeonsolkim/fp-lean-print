# Contributing

Thanks for helping improve this print layout.

Useful contributions include:

- fixes for generated bundle compatibility
- clearer setup instructions
- screenshots or sample output notes
- print CSS adjustments that preserve the Mathematics in Lean-like layout goal

When proposing a change, please include:

- the source bundle or generated HTML version you tested with
- your browser/PDF renderer
- before/after screenshots when the change is visual

This repository does not host the original book content, so please avoid adding
large generated bundles or copyrighted book artifacts directly to the repo.
